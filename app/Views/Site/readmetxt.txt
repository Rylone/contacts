API CONTACTS MANAGER


Description
Cette API est un gestionnaire de contacts. Il est encodé en UTF-8. 
Auteur : Priscillia HILLION

Quelles sont ses fonctionnalités ?

 - Lister les contacts 
 - Editer les contacts

 - Mettre des contacts en favoris
